# Flask Notification System

A notification management system built with Flask.

## Features
- Add notifications
- Mark as read
- Delete notifications
- SQLite database
- Dashboard-style interface

## Technologies Used
- Python
- Flask
- SQLite3
- HTML

## Installation

pip install flask

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Purpose
focuses on dashboard-style backend features and notification systems.
